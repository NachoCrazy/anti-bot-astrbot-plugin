# Anti-Bot Plugin (反机器人插件)

这是一个专为 AstrBot 设计的插件，用于自动检测并"怼回"那些提及"机器人"相关关键词的消息。

## 功能特点

- 自动检测包含"bot"、"机器人"等关键词的消息
- 区分普通消息和@消息，提供不同的回复内容
- 可自定义检测正则表达式和回复内容
- 防止自我触发机制
- 可开关的表情符号增强功能

## 配置选项

安装后可通过 AstrBot 管理面板进行以下配置：

- `enabled`: 是否开启反机器人检测（默认: true）
- `detect_regex`: 触发检测的正则表达式（默认: `(?i).*(bot|机器人|机机人|你是[个個]?机器|bot.?bot|机器人.?机器人).*`）
- `reply_text`: 普通触发回复（默认: "你才是机器人baka！！！😡"）
- `at_reply_text`: 被@时回复（默认: "你@我干嘛！有种再说一遍？你才是机器人baka！！！💢"）
- `add_emoji`: 自动加随机表情（默认: true）

## 使用说明

1. 将插件文件夹放入 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot
3. 在管理面板中配置相关参数（可选）
4. 插件会自动检测并回复含有机器人相关关键词的消息

## 注意事项

- 此插件专为 NapCat 平台优化设计
- 可通过配置完全自定义检测规则和回复内容
- 具有防自触发机制，避免无限循环回复
